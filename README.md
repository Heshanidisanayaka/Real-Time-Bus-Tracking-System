# Real-Time Bus Tracking System

A modern web application for tracking city buses in real time, managing routes, ticketing, and providing passenger services.

## Features

- **Live bus tracking** with map visualization
- **Multilingual support** (English, Sinhala, Tamil)
- **Voice announcements** for next stop and arrival alerts
- **SOS Safety Feature** – emergency contact notification and live location sharing
- **Lost & Found System** – report and track lost items
- **Online payments** integration for ticket purchases
- **Student bus pass** management

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Heshanidisanayaka/Real-Time-Bus-Tracking-System.git
cd Real-Time-Bus-Tracking-System

# Install dependencies
npm install

# Run the development server
npm run dev
```
